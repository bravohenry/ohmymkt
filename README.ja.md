# ohmymkt

マーケティング実行向けに最適化した OpenCode プラグインワークスペースです。

主要トポロジーは以下です。

- Primary: `growth-manager`
- Planning: `requirements-analyst`, `plan-reviewer`
- Execution: `execution-manager`
- Domain specialists: AEO / Content Ops / Writer / Growth Analyst / Research / SEO
- Runtime tools: 18 `ohmymkt_*` tools
- State directory: `.ohmymkt/`
